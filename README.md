# Pushkar Talwalkar — Portfolio (Vite + React + Tailwind)

This is a lightweight Vite + React + Tailwind project containing a single-page portfolio for Pushkar Talwalkar.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
npm run preview
```

### Notes
- Tailwind is pre-configured (tailwind.config.cjs, postcss.config.cjs).
- The résumé `PushkarTalwalkar.pdf` is included in `public/`.
- If you don't have Node 18+, please install a compatible Node version.
